Context & Goal

The research aimed to understand how real-time data (e.g., bin fullness) and peer coordination could influence recycling behavior, reduce contamination, and prevent waste overflow. The digital interface was one of several interventions tested in the field.

Final refined UI on mobile.

My Contributions

  • Co-ran design workshops and helped analyze interview data
  • Designed the interface and user journey in Figma
  • Developed the web front end using JavaScript, HTML, and CSS
  • Used Git for version control and backup
  • Mounted sensors and supported data connection to the interface

Research to Design

Workshops highlighted two main challenges: (1) lack of visibility into bin status across different recycling rooms, causing unnecessary trips or overflow, and (2) difficulty transporting bulky waste to the larger recycling center, which was only accessible by car. These findings directly shaped the application's feature set.
